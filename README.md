# 元火弹幕姬 Light

实时桌面弹幕服务，使用 tauri 重写。

## 使用方法

启动后，会在本地的指定端口开启 websocket 服务，接收来自客户端的弹幕数据（`json` 格式）。

```json
{
  "text": "弹幕测试",
  "size": 25,
  "color": "#ff0000"
}
```

`text` 为弹幕内容，`size` 为字号（可选，单位像素），`color` 为弹幕颜色（可选，按照 css 的颜色格式）。

可以通过托盘区图标控制弹幕姬，如设置弹幕是否显示、打开配置文件、退出弹幕姬等。
